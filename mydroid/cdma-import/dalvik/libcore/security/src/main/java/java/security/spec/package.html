This package provides the classes and interfaces needed to specify keys and parameter for encryption and signing algorithms. Following standards are supported: (1) PKCS#1 RSA encryption standard; (2) FIPS-186 DSA (signature) standard; (3) PKCS#8 private key information standard. Keys may be specified via algorithm or in a more abstract and general way with ASN.1. The parameters for the Elliptic Curve (EC) encryption algorithm are only specified as input parameters to the relevant EC-generator.
