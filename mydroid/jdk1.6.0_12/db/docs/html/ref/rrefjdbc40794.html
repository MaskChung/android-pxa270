java.sql.Statement interface

Derby does not implement the JDBC 1.2 setEscapeProcessing method of java.sql.Statement. In addition, the following optional methods raise "Feature not supported" exceptions:

  • cancel( )
  • execute( java.lang.String, int[] )
  • execute( java.lang.String, String[] )
  • executeUpdate( java.lang.String, int[] )
  • executeUpdate( java.lang.String, String[] )
  • ResultSet objects
